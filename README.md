# README
##这是基于andriod的V2EX手机APP的开发不同进度的文件，可帮助你在手机上浏览V2EX
###                             Author：wineoffree
####使用说明 
导入andriod studio即可使用
####主要功能
#####提供v2ex网页11个tab的展示界面
可通过手势翻页或点击相应TAB进入相应界面

#####提供刷新功能
可通过下拉刷新或手动点击MENU里的刷新功能

#####提供搜索功能
提供两种搜索，一种通过用户名搜索相应用户<br>
一种通过输入关键词查询全部TAB中的内容

#####无网显示
由于本地缓存至文件，可以不用网络显示加载好的内容

#####提供删除本地文件缓存
主界面图片文字会本地缓存占用空间，可删除

#####提供自定义分页
可以根据个人习惯调整TAB顺序
